# LUMIÈRE — Essence of Luxury

![LUMIÈRE Banner](https://raw.githubusercontent.com/panduthegang/Lumiere/main/public/Thumbnail.jpg)

LUMIÈRE is a premium, high-end fragrance boutique web application designed to provide an immersive olfactory experience. Built with a focus on minimalist luxury, the platform showcases fine perfumes through sophisticated animations, elegant typography, and a "less is more" design philosophy.

---

## 💎 Project Essence

- **Brand Identity**: Parisian-inspired luxury with a focus on Grasse craftsmanship.
- **Design Philosophy**: Minimalist, high-contrast, and motion-heavy.
- **Visuals**: Large-scale imagery, glassmorphism, and editorial typography.

## ✨ Key Features

- **Immersive Hero Section**: Featuring a subtle parallax effect and smooth typography entrance animations.
- **Minimalist Navbar**: A dynamic, glassmorphic header that simplifies on scroll and features a unique brand "shimmer" accent.
- **Asymmetrical Gallery**: A craftsmanship section that breaks standard grid layouts for a more organic, artistic feel.
- **Infinite Testimonials Marquee**: A continuous, high-performance press recognition scroll that pauses on interaction.
- **Product Discovery**: Interactive collection grid with sophisticated lift-and-scale hover effects.
- **Editorial Typography**: Leverages *Cormorant Garamond* and *Montserrat* for a classic-meets-modern aesthetic.

## 🚀 Setup Guide

To run LUMIÈRE locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/panduthegang/Lumiere.git
   cd Lumiere
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
